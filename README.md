# Golden Hour Rides

Simple 3D open-world cab + plane sandbox built with Next.js and Three.js.

## Run Locally

```bash
pnpm install
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000).

Optional debug mode:

```text
http://localhost:3000/?debug=1
```

## Controls

- `W / S` or `Arrow Up / Down`: accelerate / reverse
- `A / D` or `Arrow Left / Right`: steer
- `Shift`: speed boost
- `1`: switch to cab
- `2`: switch to plane
- `Q`: plane lift / climb
- `E`: plane descend

## How To Play

1. Drive to the pickup marker.
2. Stop near the passenger so boarding starts.
3. Follow the top navigation + minimap to the dropoff marker.
4. Complete rides to earn money and improve rating.

Tip: use cab for tighter city movement, plane for longer routes.
